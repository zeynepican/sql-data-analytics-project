# SQL Data Analytics Project

This repository contains a collection of SQL scripts focused on analytical problem-solving and data exploration. The project was developed as part of a hands-on learning process by following the *DataWithBaraa* SQL Data Analytics tutorial series.

##  About the Project

This project is designed to strengthen practical SQL skills in the context of data analytics. Rather than being a standalone original project, it reflects a structured learning journey where analytical concepts are implemented step by step.

Throughout this repository, various SQL techniques are applied to analyze and interpret data within a relational database environment.

##  Scope of Analysis

The scripts included in this project cover a wide range of analytical patterns, including:

- Database exploration
- Measures and key metrics calculation
- Time-based trend analysis
- Cumulative analytics
- Data segmentation
- Part-to-whole analysis
- Performance analysis

Each script focuses on a specific analytical concept and demonstrates how SQL can be used effectively for data-driven decision-making.

##  Key Objectives

- Improve SQL query writing skills for analytics
- Understand common data analysis patterns
- Apply best practices in SQL
- Gain hands-on experience with real-world analytical scenarios

##  Learning Context

This project was developed by following the tutorial series created by **DataWithBaraa**. The goal is to reinforce theoretical knowledge through practical implementation and build a solid foundation in data analytics using SQL.

##  Notes

- All queries are written with clarity and readability in mind
- The project emphasizes understanding over complexity
- It serves as a reference for common analytical SQL patterns
